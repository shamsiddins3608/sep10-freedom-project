## Learning Log 1
### 4/15/26
* Aframe is a very cool open-source web framework for building virtual reality and augmented reality experiences. I tinkered with it by watching this [Playlist](https://www.youtube.com/playlist?list=PLWkWuhMLkR7D_VSEMkj45NIgF8i2dlUce) on youtube which teaches the fundementals, and elements of aframe.

## Learning Log 2
### 4/16/26
* I added aframe to my project, and added a red cube. I read more about aframe on this [Guide](https://aframe.io/docs/1.5.0/introduction/) and learned that A-Frame is optimized from the ground up for WebVR. While A-Frame uses the DOM, its elements don’t touch the browser layout engine.

## Learning Log 3
### 4/17/26
* I looked at this [Presentation](https://docs.google.com/presentation/d/1C8xDyvXzVgtX3G8UdXVNEn2PWVnsLF0sPu4QMRCL7xY/edit?slide=id.g849c2abd37_0_0#slide=id.g849c2abd37_0_0) and learned how to position, changed color and size of an object. I also learned that Components have a src attribute that enables you to specify an image to use as the texture for the surface of the component.

## Learning Log 4
